# POSLab

POSLab is a modular Point-of-Sale (POS) simulator written in C for Linux.

The project is designed to simulate the architecture of a real payment terminal, including transaction processing, ISO 8583 messaging, storage, reporting, cryptography, and host communication.

## Features

- Modular architecture
- ISO 8583 message builder/parser
- Transaction engine
- Report engine
- Configuration manager
- Logger
- Host simulator
- Console interface
- Qt interface
